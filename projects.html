Design Thinking Project

AutoTreats - Smart Vending Machine

AutoTreats is a smart vending machine concept developed using Design Thinking methodology. The project involved understanding user needs, ideation, prototyping, and testing. We created both digital wireframes in Figma and a physical cardboard prototype.

Key Features:

  • Touchscreen interface for easy selection
  • Cashless payment options (QR, card, e-wallet)
  • Real-time inventory tracking
  • Personalized recommendations
  • Energy-efficient design

MHuat - Financial Management App

MHuat is a financial management app developed during a hackathon competition. The app helps users track their expenses, set savings goals, and manage their personal finances effectively. Built using Dart and Flutter in Android Studio.

Key Features:

  • Expense tracking and categorization
  • Savings goal setting
  • Monthly budget planning
  • Spending insights and reports
  • Secure data storage
